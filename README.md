# 画面キャプチャツールの使用方法

試験の公正性を保つため，試験中の画面を定期的に保存します．以下の手順に従って実行してください．

## 1. ツールのダウンロードと準備
1. 指定されたURLから，このGitHubページを開きます．
2. 画面右上にある緑色の **「<> Code」** というボタンをクリックし，表示されたメニューから **「Download ZIP」** を選択してダウンロードします．
3. ダウンロードしたZIPファイルを右クリックし，**「すべて展開」**を選択してフォルダを開きます．
4. 展開されたフォルダの中に `start_capture.bat` と `capture.ps1` があることを確認してください．

## 2. 実行手順
1. 試験開始直前に **`start_capture.bat`** をダブルクリックします．
2. 「WindowsによってPCが保護されました」と表示された場合は，**「詳細情報」**をクリックし，**「実行」**を選択してください．
3. 黒い画面が立ち上がり，「画面キャプチャツールを実行しています」と表示されれば成功です．
4. **試験終了まで，この黒い画面は閉じないでください．**（最小化して裏で動かしておくのは問題ありません）

## 3. 試験終了後の対応
1. 試験が終了したら，黒い画面の右上にある**「×」ボタン**を押して終了してください．
2. 保存された画像は **`C:\capture`** フォルダ内にあります．
3. 指示に従い，解答ファイル（.cpp）と共に，このフォルダ内の画像ファイルを提出してください．

## 4. トラブル時の連絡
実行時にエラーが出る，またはPCの動作が著しく重くなった場合は，すぐに試験監督（TA）へ報告してください．
