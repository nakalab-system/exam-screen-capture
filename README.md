# 画面キャプチャツールの使用方法

試験の公正性を保つため，試験中の画面を定期的に保存します．以下の手順に従って実行してください．

## 1. ツールのダウンロードと準備
1. 指定されたURLから，このGitHubページを開きます．
2. 画面右上にある緑色の **「<> Code」** というボタンをクリックし，表示されたメニューから **「Download ZIP」** を選択してダウンロードします．
3. ダウンロードしたZIPファイルを右クリックし，**「すべて展開」**を選択してフォルダを開きます．
4. 展開されたフォルダの中に以下のファイルがあることを確認してください．
   - `1_start_hidden.bat`
   - `2_check_status.bat`
   - `3_stop.bat`
   - `capture.ps1`

## 2. 実行手順
1. 試験開始直前に **`1_start_hidden.bat`** をダブルクリックします．
2. 「WindowsによってPCが保護されました」と表示された場合は，**「詳細情報」**をクリックし，**「実行」**を選択してください．
3. 黒い画面が立ち上がり，「Starting Background Screen Capture」と表示されたら，数秒後に自動で閉じます．これで画面のキャプチャが裏で開始されます．
4. キャプチャが正常に動作しているか確認したい場合は，**`2_check_status.bat`** をダブルクリックしてください．「[RUNNING] The tool is currently active.」と表示されていれば正常に動作しています．

## 3. 試験終了後の対応
1. 試験が終了したら，**`3_stop.bat`** をダブルクリックしてキャプチャを停止してください．
2. 保存された画像は **`C:\capture`** フォルダ内にあります．
3. 保存された画像を確認し，ctrl+Aで全て選択し，ZIP形式で圧縮して提出してください．

## 4. トラブル時の連絡
実行時にエラーが出る，またはPCの動作が著しく重くなった場合は，すぐに試験監督（TA）へ報告してください．
